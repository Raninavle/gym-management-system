workout regimen:

workout regimen can only be created based on BMI range. BMI stands for body mass index.Body Mass Index (BMI) is a person's weight in kilograms (or pounds) divided by the square of height in meters (or feet)

    BMI<= 18.5 set following as a default workout regimen:
  • Monday : chest
  • Tuesday : biceps

  • Thursday : back
  • Friday : triceps
  • Saturday : rest
  • Sunday : rest
